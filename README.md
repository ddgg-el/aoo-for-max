Aoo for Max8
===========

### Sviluppo

Il progetto dipende da [Aoo](https://aoo.iem.sh/) e dalla [max-sdk](https://github.com/Cycling74/max-sdk?tab=readme-ov-file) che sono inclusi come sottomoduli del progetto. 

### Build instruction
```bash
$ mkidr build
$ cd build
$ cmake ..
$ cmake --build . 
```
